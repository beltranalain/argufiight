# OG Images Required

This directory should contain the following images for social media sharing:

## Required Images

1. **og-image.png**
   - Dimensions: 1200x630 pixels
   - Format: PNG
   - Purpose: Open Graph image for Facebook, LinkedIn, etc.
   - Content: Should include Argufight branding, tagline, and visual elements
   - Location: `/public/og-image.png`
   - Referenced in: `app/page.tsx` and other pages

2. **twitter-card.png**
   - Dimensions: 1200x628 pixels (or 1200x630)
   - Format: PNG
   - Purpose: Twitter Card image
   - Content: Similar to og-image but optimized for Twitter
   - Location: `/public/twitter-card.png`
   - Referenced in: `app/page.tsx` and other pages

## Design Guidelines

- Use Argufight brand colors (electric blue, purple gradient)
- Include the logo and tagline
- Make text readable at small sizes
- Keep important content in the center (safe zone)
- Use high contrast for readability

## Tools for Creation

- Canva (templates available)
- Figma
- Adobe Photoshop/Illustrator
- Online tools like Bannerbear or Cloudinary
